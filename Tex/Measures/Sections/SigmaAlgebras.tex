\begin{definition}
A $\sigma$-algebra, $\m{A}$, on a set $X$ is a collection of subsets of $X$ satisfying the following three axioms:
\begin{itemize}
\item $X \in \m{A}$
\item If $X \in \m{A}$, then $X^c \in \m{A}$
\item If $A_1, A_2, \dots \in \m{A}$, then $\bigcup_{i \in \N} A_i \in \m{A}$
\end{itemize}
\end{definition}
